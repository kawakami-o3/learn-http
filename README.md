# learn-http

Run server

```
% cargo run --bin http-server
```

Run client

```
% cargo run --bin http-client
```

# Features

* Configuration
  * [ ] Content root
  * [x] Host's IP address
  * [x] Listen port
  * [x] SERVER header
  * [ ] Extension status code
  * [ ] Location response header
  * [ ] Allow response header
  
* [ ] WWW-authenticate
